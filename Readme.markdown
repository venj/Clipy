# Clipy

![Icon](icon.png)

Clipboard history management app for Windows.

Main features are basically usable. Settings pane and keyboard shortcuts are not available yet. 

![Screenshot](screenshot.png)

## Installation

### Using binary release:

1. Download "[Clipy_1.0.4-release.zip](https://github.com/venj/Clipy/releases/download/1.0.4/Clipy_1.0.4-release.zip)" from [releases](https://github.com/venj/Clipy/releases) page. Unzip and install the app.

2. Enjoy and file bugs.

### From source code:

Clone this project and build with Visual Studio Community 2015.
